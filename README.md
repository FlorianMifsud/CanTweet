# Can Tweet

Check if you can tweet, 2 lines.

```javascript
import { canTweet } from "@mrlerandom/can-tweet";
const tweet = `Hello World ! 👩‍👩‍👧‍👦👩‍👩‍👧‍👦 你好世界 ！Im https://www.github.com/florianmifsud`;
console.log(canTweet(tweet)); //return true
```

✅ 0 dependencies

✅ With emoji

✅ With URL
